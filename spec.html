An API playground.

Specifications

  1. No copyrighted content can be used in this project (including video, audio, images, etc.)
  2. One API token can be used for all available services
  3. The following are the only REST calls the service can provide to users: GET, POST
  4. All content must be ephemeral and be automatically deleted after the 10 most recent posts per API token
  5. Any content passed through the API cannot exceed 1 MB per request
  6. All code for the service must be open source
  7. If any of the services violate the rules listed above, they will be removed from the central hub
