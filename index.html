An API playground.

Build small mutations with everyone.

This project is an attempt to encourage developers of all skills to build small, focused services with only two API calls.

Read about the concept.

API Specs

Specifications can be viewed in spec.html

Current script flow:

  1. Grab token displayed on your screen.
  2. Add 1-4 services to your script.
  3. Play! Or share output however you want.

What kind of services can be built?

Imagine uploading a jpeg to three external services - one creates glitch artifacts, the next one converts it into animated gif and the next one sends it out to a Twitter account.
